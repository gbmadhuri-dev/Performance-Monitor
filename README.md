# API Performance Monitor

A web-based application that allows users to test any API or website URL and measure its response time.

## Features
- Measures response time in milliseconds
- Displays HTTP status code
- Shows timestamp of each request
- Stores history of past API tests

## How It Works
1. Enter an API or website URL
2. Click on "Test API"
3. View response time, status code, and timestamp
